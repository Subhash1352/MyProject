# IoT Based Health Monitoring Device for Elderly People Using ESP32

## Project Overview
This project is an IoT-based health monitoring system designed for elderly individuals. The system continuously monitors vital health parameters such as heart rate, SpO2, temperature, pressure, and stress levels.

The collected data is processed using the ESP32 microcontroller and transmitted to the Blynk IoT platform via WiFi using the MQTT protocol. Caregivers and doctors can monitor patient health remotely through a mobile dashboard.

If any health parameter exceeds predefined threshold values, the system automatically sends an email alert to the doctor or caretaker.

---

## Features
- Real-time health monitoring
- Remote monitoring via IoT cloud
- Automatic alert system
- Low-cost and portable solution
- Continuous monitoring of elderly patients

## Hardware Components
- ESP32 Development Board
- BMP180 Temperature & Pressure Sensor
- MAX30102 Heart Rate & SpO2 Sensor
- GSR Stress Sensor
- Breadboard
- Jumper Wires
- Power Supply

---

## Software Used
- Arduino IDE
- ESP32 Board Package
- Blynk IoT Platform
- MQTT Protocol

---

## System Workflow
1. Sensors collect physiological data
2. ESP32 processes sensor values
3. Data is transmitted to Blynk IoT cloud
4. Users monitor data through mobile dashboard
5. Alerts are triggered if abnormal conditions occur
